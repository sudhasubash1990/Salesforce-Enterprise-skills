## Practice Exercises

1. **Repository extension** — Add a new knowledge article following build standards; pass validate_repository
2. **Industry pack design** — Outline a new industry scenario pack with cross-links and metadata
3. **Maturity program** — Design an organizational adoption plan using ba-maturity-model Sections 14–15
4. **AI coaching playbook** — Document how teams should use the repository with Cursor for a 12-week program
5. **ADR authorship** — Propose and document a design decision in implementation/design-decisions.md

## Exit Checklist

- [ ] Shapes enterprise BA strategy and repository evolution
- [ ] Authors or approves repository extensions per build standards
- [ ] Coaches teams on AI-assisted consulting with anti-hallucination discipline
- [ ] Governs knowledge lifecycle (draft → approved → maintenance)
- [ ] Recognized as trusted advisor for enterprise Salesforce transformation
- [ ] Contributes to ROADMAP and organizational capability metrics
