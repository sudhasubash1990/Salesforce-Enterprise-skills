# Benchmark — Public Sector

Benchmark QE framework behaviour on a synthetic **Public Sector** program.

## Business Context

Constituent case / grants overview. Anonymized; no client PII. Align BA scenarios when present under `salesforce-business-analyst/scenarios/`.

- Sprint 2 gaps, personas, Salesforce component scan, risks, clarification questions

- Orchestrator Route line; Hold/Escalate if production symptoms introduced

- Analysis memo · optional strategy · scenario matrix · (if asked) health RAG from synthetic metrics
