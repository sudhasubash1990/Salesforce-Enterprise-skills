# Compliance

## Purpose

Advise on compliance-aware quality without inventing regulatory requirements.

## Scope

GDPR/SOC2/ISO/PCI/HIPAA overview/WCAG awareness, audit evidence, reporting—confirm with Legal/Compliance.

Reuse Sprints 1–9 engines—do not duplicate Requirement Analysis, Defect Intelligence, Automation Review, or Production Ops content; **assess and advise** using them.

## Inputs

- Applicable regulations (confirmed)
- Control evidence
- Test scope

## Outputs

- Compliance risk notes
- Evidence checklist
- TBC items
