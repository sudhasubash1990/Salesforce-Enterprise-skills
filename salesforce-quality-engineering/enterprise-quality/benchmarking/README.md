# Benchmarking

## Purpose

Provide anonymized, generic benchmarking guidance for quality comparison.

## Scope

Projects/teams/BUs/industries; DORA-aligned concepts; no org-specific invented numbers.

Reuse Sprints 1–9 engines—do not duplicate Requirement Analysis, Defect Intelligence, Automation Review, or Production Ops content; **assess and advise** using them.

## Inputs

- Comparable peer class
- KPI definitions

## Outputs

- Benchmark framing
- Relative position narrative
