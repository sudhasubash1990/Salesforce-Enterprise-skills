# Portfolio Management

## Purpose

Govern quality across multi-project Salesforce portfolios.

## Scope

Cross-project metrics, heat maps, enterprise risk, strategic recommendations.

Reuse Sprints 1–9 engines—do not duplicate Requirement Analysis, Defect Intelligence, Automation Review, or Production Ops content; **assess and advise** using them.

## Inputs

- Multi-project scorecards
- Release calendars
- Shared risks

## Outputs

- Portfolio dashboard design
- Heat maps
- Strategic actions
