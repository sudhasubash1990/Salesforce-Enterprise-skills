# Project Health

## Purpose

Score and narrate Salesforce project quality and delivery health for executives.

## Scope

Requirements through stakeholder risk, RAG status, executive summary, recommendations.

Reuse Sprints 1–9 engines—do not duplicate Requirement Analysis, Defect Intelligence, Automation Review, or Production Ops content; **assess and advise** using them.

## Inputs

- Requirements/AC evidence
- Coverage/defect/release signals
- RAID

## Outputs

- Health scorecard
- Traffic lights
- Executive summary
