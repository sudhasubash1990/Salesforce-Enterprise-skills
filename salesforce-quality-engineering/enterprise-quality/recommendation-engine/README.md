# Strategic Recommendation Engine

## Purpose

Generate prioritized enterprise recommendations from multi-signal evidence.

## Scope

Quality/defect/automation/architecture/ops/compliance signals → ranked actions.

Reuse Sprints 1–9 engines—do not duplicate Requirement Analysis, Defect Intelligence, Automation Review, or Production Ops content; **assess and advise** using them.

## Inputs

- Scorecards
- Trends
- Risks
- Constraints

## Outputs

- Prioritized backlog
- Effort vs value view
