# Governance

## Purpose

DoR/DoD, workflows, quality gates, and audit readiness for ADO-based delivery.

## When Used

Process setup and every sprint/release gate.

## Inputs

- Methodology (Agile/Scrum/SAFe)
- Org standards

## Outputs

- Gate checklists
- Workflow expectations

## Best Practices

- Keep gates evidence-based
- Align with Sprint 5 checklists
- Audit: who/when/what linked
