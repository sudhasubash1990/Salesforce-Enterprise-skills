# Epics

## Purpose

Portfolio-level outcomes that span multiple features for Salesforce programs.

## When Used

Program initiation, PI planning, or when a large business outcome needs decomposition.

## Inputs

- Business outcomes
- Roadmap themes
- Cloud/industry context

## Outputs

- Epic work items
- Child Feature links

## Best Practices

- Outcome-oriented titles
- Measurable success criteria
- Link Features; avoid task-level detail

| [Epic](epic.md) | Epic work item guidance |

## Scope note

**Sprint 6 focuses on Delivery Intelligence** — work item quality, relationships, and ADO-ready artifact content.  
**Not in scope:** Azure DevOps REST API automation / auto-publish (optional later enhancement).
