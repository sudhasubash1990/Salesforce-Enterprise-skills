# Bugs

## Purpose

Defect work items with enterprise repro quality and release impact.

## When Used

When verified failures are found in any environment.

## Inputs

- Failing case/scenario
- Environment/build
- Evidence

## Outputs

- Bug work item
- Links to Story/Case
- Retest notes

## Best Practices

- Severity ≠ Priority
- Always include Expected vs Actual
- Assess regression impact

| Document | Focus |
