# Playbooks

## Purpose

Reusable facilitation playbooks for core QE ceremonies and gates.

## Scope

Sprint 5 — process playbooks (not BA OCM playbooks; cross-link BA where needed).

## Available Documents

| Playbook | Focus |
|----------|-------|
| [Requirement Review](requirement-review.md) | Facilitate QE requirement review using the Requirement Analysis Engine. |
| [Story Grooming](story-grooming.md) | QE participation in grooming to improve testability of stories and AC. |
| [Test Planning](test-planning.md) | Produce or refresh Test Plan from strategy and backlog. |
| [Test Design Review](test-design-review.md) | Peer review of scenarios/cases/coverage before execution. |
| [Regression Planning](regression-planning.md) | Select risk-based regression scope for a release. |
| [Defect Triage](defect-triage.md) | Triage defects for severity, priority, owner, and target fix. |
| [Release Readiness](release-readiness.md) | Assemble readiness evidence and residual risk for Go/No-Go. |
| [Production Validation](production-validation.md) | Execute production validation / early verification after deploy. |
| [Hypercare](hypercare.md) | Operate hypercare quality checks and feedback loop. |
| [Knowledge Transfer](knowledge-transfer.md) | Transfer QA knowledge, packs, and monitoring cues to BAU. |
