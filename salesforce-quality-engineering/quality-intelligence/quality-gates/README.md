# Quality Gates (Intelligence)

## Purpose

## Scope

Complements Sprint 5 checklists and ado/governance—focus on decision logic.

## Inputs

- Open defects
- Metrics
- Readiness checklists

## Outputs

- Gate pass/warn/fail recommendations
